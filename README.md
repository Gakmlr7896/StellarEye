# Deep-Learning-Fusion-for-Stellar-Age-and-Metallicity-
A deep learning model to predict stellars' and metallicity 

*This project is for researching purposes and its not meant to be commercial.*

Deep Learning Fusion for stellar Age and Metallicity is a project the uses deep learning models and neural networks to estimate the stellars' age and metallicity, more informations will be added soon.
